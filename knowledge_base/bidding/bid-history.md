# Viewing Bid History

Track your bidding activity and see auction bid history.

## Auction Bid History

- **On listing page** - See all bids on that auction
- **Bid amounts** - See how bidding progressed
- **Bidder usernames** - See who's bidding (privacy protected)
- **Timestamps** - When each bid was placed

## What You Can See

- **Your bids** - All bids you've placed
- **Current high bid** - Who's currently winning
- **Bid increments** - How much each bid increased
- **Time remaining** - How much time left in auction

## Privacy

- **Usernames only** - Real names not shown
- **Your bids visible** - Others can see your username if you bid
- **Private details** - Personal information never shared

## Using Bid History

- **Track your activity** - See all auctions you've bid on
- **Monitor progress** - Check if you're still high bidder
- **Learn patterns** - See when others typically bid
- **Plan strategy** - Use history to inform future bids

## Tips

- **Check regularly** - Stay updated on your bids
- **Review before bidding** - See current bid activity
- **Use for strategy** - Learn from past bidding patterns
- **Keep organized** - Track multiple auctions easily
