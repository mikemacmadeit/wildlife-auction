# Account Security Guide

How to keep your Wildlife Exchange account secure and protected.

## Password Security

### Creating a Strong Password

**Password requirements:**
- At least 8 characters long
- Mix of letters and numbers
- Uppercase and lowercase letters
- Special characters recommended
- Unique (not used on other sites)

**Good password examples:**
- ✅ "HorseRanch2024!Secure"
- ✅ "Wildlife@Exchange#2024"
- ❌ "password123"
- ❌ "12345678"
- ❌ Your name or email

**Password tips:**
- Use a password manager
- Don't reuse passwords
- Change password regularly
- Never share your password
- Use two-factor authentication if available

### Password Management

**Use a password manager:**
- Stores passwords securely
- Generates strong passwords
- Auto-fills login forms
- Syncs across devices
- Prevents password reuse

**Popular options:**
- LastPass
- 1Password
- Bitwarden
- Dashlane
- Built-in browser password managers

## Account Protection

### Email Verification

**Why verify email:**
- Required for purchases and listings
- Enables password reset
- Receives important notifications
- Account recovery option
- Security verification

**Keep email secure:**
- Use strong email password
- Enable two-factor on email
- Don't share email password
- Monitor email for suspicious activity
- Update email if compromised

### Two-Factor Authentication

**What is 2FA:**
- Extra security layer
- Requires password + code
- Code from phone or app
- Prevents unauthorized access
- Highly recommended

**How to enable (if available):**
1. Go to Account Settings
2. Find "Security" or "Two-Factor Authentication"
3. Follow setup instructions
4. Verify with phone or app
5. Save backup codes

## Recognizing Threats

### Phishing Attempts

**What is phishing:**
- Fake emails or messages
- Pretend to be from Wildlife Exchange
- Ask for password or personal info
- Include suspicious links
- Try to steal your account

**How to recognize:**
- Suspicious sender email
- Urgent or threatening language
- Asking for password
- Links to fake websites
- Poor grammar or spelling

**What to do:**
- Don't click suspicious links
- Don't enter password on suspicious sites
- Report to support
- Verify sender is legitimate
- When in doubt, contact support directly

### Suspicious Activity

**Signs of compromise:**
- Can't sign in (password changed)
- Unfamiliar orders or listings
- Emails about account changes you didn't make
- Unusual activity in account
- Notifications about unknown actions

**If account is compromised:**
1. Try to sign in immediately
2. Use password reset if needed
3. Change password to strong new one
4. Check account for unauthorized activity
5. Contact support immediately
6. Review and secure email account

## Best Practices

### Daily Security Habits

**Protect your account:**
- Use strong, unique password
- Don't share login credentials
- Log out on shared computers
- Keep browser updated
- Use secure internet connection

**Monitor your account:**
- Check account regularly
- Review orders and activity
- Watch for suspicious emails
- Report issues immediately
- Keep contact info updated

### Public Computer Safety

**If using public computer:**
- Never save passwords
- Always log out when done
- Clear browser history
- Don't access sensitive info
- Use incognito/private mode

**Best practice:**
- Avoid public computers for account access
- Use your own device when possible
- Use mobile app if available
- Be extra cautious on shared devices

## Recovery and Support

### If You're Locked Out

**Password reset:**
1. Go to sign-in page
2. Click "Forgot Password"
3. Enter your email
4. Check email for reset link
5. Create new strong password
6. Sign in with new password

**If password reset doesn't work:**
- Check spam folder for email
- Wait a few minutes and try again
- Verify email address is correct
- Contact support for help
- Provide account information

### Contacting Support for Security Issues

**When to contact:**
- Account compromised
- Suspicious activity
- Can't access account
- Unauthorized changes
- Security concerns

**What to include:**
- Your email address
- Description of issue
- When it happened
- Any suspicious activity
- Screenshots if available

## Additional Security Tips

### Device Security

**Secure your devices:**
- Use device passwords/PINs
- Keep software updated
- Use antivirus software
- Enable device encryption
- Lock device when not in use

### Network Security

**Secure connections:**
- Use trusted Wi-Fi networks
- Avoid public Wi-Fi for sensitive actions
- Use VPN if on public network
- Check for secure connection (HTTPS)
- Verify network is legitimate

### Information Sharing

**What NOT to share:**
- Your password (never, ever)
- Account verification codes
- Personal security questions
- Payment information
- Account access details

**Safe to share:**
- Your name (public profile)
- Location (for listings)
- Business information (if selling)
- General contact info (if you choose)
