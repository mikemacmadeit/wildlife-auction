# Buyer vs Seller Accounts

Understanding the difference between buyer and seller accounts.

## Account Types

### Buyer Account
- **For purchasing** - Buy items from sellers
- **Basic features** - Browse, bid, purchase
- **Free to use** - No fees to buy
- **Simple setup** - Just create account and verify email

### Seller Account
- **For selling** - List and sell items
- **Additional features** - Create listings, manage sales
- **Payment setup** - Must connect payment account
- **Verification** - May require additional verification

## Can I Be Both?

- **Yes!** - Same account can buy and sell
- **No separate accounts needed** - One account does both
- **Switch easily** - Use buyer features or seller features as needed
- **Unified dashboard** - See both purchases and sales

## Buyer Features

- Browse and search listings
- Place bids on auctions
- Make offers on fixed-price listings
- Purchase items
- Track orders
- Contact sellers
- Use watchlist

## Seller Features

- Create listings
- Manage inventory
- Respond to bids and offers
- Process orders
- Receive payouts
- View sales analytics
- Manage seller profile

## Account Setup

### For Buying
1. Create account
2. Verify email
3. Start browsing and buying

### For Selling
1. Create account
2. Verify email
3. Complete seller profile
4. Set up payment account
5. Start creating listings

## Switching Roles

- **No switching needed** - Account works for both
- **Access all features** - Buyer and seller tools available
- **Unified experience** - One dashboard for everything
