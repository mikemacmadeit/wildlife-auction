# Browsing and Searching Listings

Learn how to find what you're looking for on Agchange.

## Browsing Listings

- **Browse by category** - Wildlife, Livestock, Equipment, etc.
- **Browse by location** - Find items near you
- **Browse by type** - Auctions, Fixed Price, Classifieds
- **Featured listings** - See highlighted items

## Searching

- **Search bar** - Enter keywords to search
- **Search filters** - Narrow down results
- **Save searches** - Get notified of new matches

## Filters Available

- **Category** - Filter by item type
- **Location** - Filter by state/city
- **Price range** - Set minimum and maximum
- **Listing type** - Auction, Fixed Price, etc.
- **Seller verification** - Show only verified sellers

## Search Tips

- **Use specific keywords** - More specific = better results
- **Try different terms** - If first search doesn't work
- **Use filters** - Narrow down large result sets
- **Check spelling** - Typos can affect results

## Viewing Results

- **Grid view** - See multiple listings at once
- **List view** - More details per listing
- **Sort options** - Price, date, relevance
- **Save favorites** - Add to watchlist

## Finding Specific Items

- **Search by seller** - If you know the seller
- **Search by location** - Find local items
- **Search by price** - Find items in your budget
- **Browse categories** - Explore different types
