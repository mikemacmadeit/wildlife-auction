# Compliance Review Process

Some listings require compliance review before they can be published.

## When Review Is Required

- **Regulated items** - Items requiring permits or licenses
- **Animal listings** - Wildlife, livestock, etc.
- **Certain categories** - Based on item type
- **Missing documentation** - If required docs not provided

## Review Process

1. **Submit listing** - Create and submit your listing
2. **Status changes** - Listing shows "Pending Review"
3. **Admin reviews** - Platform reviews compliance
4. **Decision made** - Approved or needs changes
5. **Notification** - You'll be notified of decision

## What's Reviewed

- **Required permits** - Proper documentation
- **Compliance with laws** - Legal requirements met
- **Accurate information** - Listing matches reality
- **Complete information** - All required fields filled

## Timeline

- **Review time** - Usually 1-3 business days
- **Complex cases** - May take longer
- **You'll be notified** - Email and in-app notification
- **Check status** - In your seller dashboard

## If Listing Needs Changes

- **You'll be notified** - What needs to be fixed
- **Make changes** - Update listing as requested
- **Resubmit** - Submit for review again
- **Faster review** - Usually quicker on resubmission

## If Listing Is Approved

- **Status changes** - Listing becomes "Active"
- **Appears in search** - Buyers can find it
- **Start selling** - Listing is live

## If Listing Is Rejected

- **Reason provided** - Why it was rejected
- **Can appeal** - Contact support if you disagree
- **Can fix and resubmit** - Address issues and try again
- **Learn from feedback** - Use to improve future listings

## Tips

- **Provide complete info** - Include all required details
- **Have permits ready** - If required for your item type
- **Be accurate** - Honest listings review faster
- **Be patient** - Reviews take time but ensure compliance
