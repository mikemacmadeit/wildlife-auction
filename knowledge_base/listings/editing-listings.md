# Editing Your Listings

You can edit your listings, but some restrictions apply depending on listing status.

## What You Can Edit

**At Any Time:**
- Description
- Photos
- Some attributes
- Contact information

**Before Bids/Offers:**
- Price or starting bid
- Reserve price
- Listing type
- Duration (for auctions)

**After Bids/Offers:**
- Price changes may be restricted
- Listing type cannot be changed
- Some fields are locked to protect buyers

2. Find the listing you want to edit
3. Click **"Edit"**
4. Make your changes
5. Click **"Save Changes"**

## Important Restrictions

- **Active auctions with bids**: Price and type cannot be changed
- **Sold listings**: Cannot be edited
- **Pending orders**: Some edits may be restricted

## Best Practices

- Review all information before publishing
- Make edits promptly if you notice errors
- Update photos if item condition changes
- Keep descriptions accurate and current

## After Editing

- Changes appear immediately on your listing
- Buyers see updated information
- You may receive notifications about edits
