# Understanding Order Status

What different order statuses mean and what to do.

## Order Statuses

### Paid
- **Payment received** - Your payment was processed
- **Order confirmed** - Order is active
- **Seller notified** - Seller knows about your order
- **Next step** - Seller arranges delivery

### In Transit
- **Item shipped** - Seller has sent the item
- **On the way** - Item is being delivered
- **Track if available** - May have tracking information
- **Wait for delivery** - Item should arrive soon

### Delivered
- **Item received** - You've confirmed receipt
- **Protection window** - Time to inspect item
- **Review item** - Check that it matches listing
- **Report issues** - If problems found

### Accepted
- **You're satisfied** - Item is as expected
- **Protection window ended** - No issues reported
- **Order complete** - Transaction finished
- **Leave review** - Optional feedback

### Disputed
- **Issue reported** - Problem with order
- **Under review** - Admin reviewing case
- **Resolution pending** - Waiting for decision
- **Check for updates** - Monitor dispute status

### Refunded
- **Refund processed** - Money returned
- **Order cancelled** - Transaction reversed
- **Check account** - Refund should appear soon
- **Contact support** - If refund delayed

## What to Do

**If status seems stuck:**
- **Check recent activity** - Look for updates
- **Contact seller** - Ask about status
- **Check email** - May have missed notifications
- **Contact support** - If status doesn't update

**If status is wrong:**
- **Verify order details** - Make sure you're looking at right order
- **Wait a moment** - Status may update shortly
- **Contact support** - If status clearly incorrect

## Tracking Orders

- **Go to "My Orders"** - See all your orders
- **Click on order** - View full details
- **Check status** - See current status
- **View messages** - Communication with seller

## Tips

- **Check regularly** - Stay updated on order status
- **Respond promptly** - If seller asks questions
- **Confirm delivery** - When you receive item
- **Report issues** - If status doesn't match reality
