# Common Errors and How to Fix Them

Quick fixes for the most common errors and issues on Agchange.

## Account and Login Errors

### "Email Not Verified"

**What it means:** You need to verify your email before you can use certain features.

**How to fix:**
1. Check your email inbox for verification link
2. Check spam/junk folder
3. Click the verification link in the email
4. If link expired, request new verification email from account settings
5. Try signing in again after verification

### "Invalid Credentials" or "Wrong Password"

**What it means:** The email or password you entered doesn't match our records.

**How to fix:**
1. Check caps lock is off (passwords are case-sensitive)
2. Verify email address is correct (check for typos)
3. Make sure password is correct
4. Try password reset if you've forgotten it
5. Clear browser cache and try again

### "User Not Found" or "Account Doesn't Exist"

**What it means:** No account exists with that email address.

**How to fix:**
1. Check email spelling is correct
2. Try different email if you have multiple
3. Make sure you completed sign-up process
4. Sign up again if you never completed registration

### "Too Many Failed Attempts"

**What it means:** You've tried to sign in too many times with wrong password.

**How to fix:**
1. Wait 15-30 minutes for lockout to clear
2. Use password reset to unlock account
3. Create new password and try again
4. Contact support if still locked out

## Payment Errors

### "Payment Declined"

**What it means:** Your payment method was declined by your bank or card issuer.

**How to fix:**
1. Check card/bank account has sufficient funds
2. Verify card details are correct (number, CVV, expiration)
3. Check card hasn't expired
4. Contact your bank to ensure no restrictions
5. Try different payment method
6. Contact support if issue persists

### "Payment Processing Failed"

**What it means:** There was an error processing your payment.

**How to fix:**
1. Check your internet connection
2. Try refreshing the page
3. Clear browser cache
4. Try different browser
5. Try different payment method
6. Wait a few minutes and try again
7. Contact support if continues

### "Payment Method Invalid"

**What it means:** The payment method you entered is not valid.

**How to fix:**
1. Check all payment details are correct
2. Make sure card hasn't expired
3. Verify billing address matches card
4. Try different payment method
5. Contact your bank to verify card is active

## Listing Errors

### "Cannot Create Listing"

**What it means:** You're missing required information or setup.

**How to fix:**
1. Make sure email is verified
2. Check payment account is connected (sellers)
3. Verify all required fields are filled
4. Check profile is complete
5. Try refreshing the page
6. Contact support if issues persist

### "Listing Not Saving"

**What it means:** There's an issue saving your listing information.

**How to fix:**
1. Check internet connection
2. Make sure all required fields are filled
3. Check for error messages on page
4. Try saving in smaller sections
5. Clear browser cache
6. Try different browser
7. Contact support if problem continues

### "Photo Upload Failed"

**What it means:** There was an error uploading your photos.

**How to fix:**
1. Check photo file size (may be too large)
2. Make sure photo format is supported (JPG, PNG)
3. Check internet connection
4. Try smaller file size
5. Try different photo
6. Clear browser cache
7. Try different browser

## Order Errors

### "Cannot Complete Checkout"

**What it means:** There's an issue completing your purchase.

**How to fix:**
1. Make sure email is verified
2. Check payment method details are correct
3. Verify sufficient funds available
4. Try different payment method
5. Clear browser cache
6. Try different browser
7. Contact support if issues persist

### "Order Not Found"

**What it means:** The order you're looking for doesn't exist or you don't have access.

**How to fix:**
1. Check you're signed in to correct account
2. Verify order number is correct
3. Check "My Orders" in dashboard
4. Make sure order wasn't cancelled
5. Contact support if order should exist

## General Errors

### "Something Went Wrong"

**What it means:** There was an unexpected error.

**How to fix:**
1. Refresh the page
2. Clear browser cache
3. Try different browser
4. Check internet connection
5. Wait a few minutes and try again
6. Contact support if error continues

### "Page Not Loading"

**What it means:** The page isn't loading properly.

**How to fix:**
1. Check internet connection
2. Refresh the page (Ctrl+F5 or Cmd+Shift+R)
3. Clear browser cache
4. Try different browser
5. Disable browser extensions
6. Try incognito/private mode
7. Contact support if problem persists

### "Feature Not Available"

**What it means:** The feature you're trying to use isn't available or you don't have access.

**How to fix:**
1. Check if feature requires account setup
2. Verify email is verified
3. Check if feature requires seller account
4. Make sure profile is complete
5. Contact support to check access

## Browser-Specific Issues

### Chrome Issues

**Common fixes:**
- Clear cache: Ctrl+Shift+Delete
- Disable extensions
- Update Chrome
- Try incognito mode
- Reset Chrome settings

### Firefox Issues

**Common fixes:**
- Clear cache: Ctrl+Shift+Delete
- Disable add-ons
- Update Firefox
- Try private browsing
- Reset Firefox

### Safari Issues

**Common fixes:**
- Clear cache: Preferences → Privacy
- Disable extensions
- Update Safari
- Try private browsing
- Reset Safari

### Edge Issues

**Common fixes:**
- Clear cache: Settings → Privacy
- Disable extensions
- Update Edge
- Try InPrivate mode
- Reset Edge

## Still Having Issues?

**If nothing works:**
1. **Contact support:**
   - Click Help button (bottom-right)
   - Select "Contact Support"
   - Fill out support ticket
   - Include error message and what you tried

2. **What to include:**
   - Exact error message
   - What you were trying to do
   - Steps you've already tried
   - Browser and device information
   - Screenshots if possible

3. **Response time:**
   - Support typically responds within 1-2 business days
   - Urgent issues may get faster response
   - Check email for replies

## Prevention Tips

**To avoid errors:**
- Keep browser updated
- Clear cache regularly
- Use supported browsers
- Keep account information updated
- Verify email immediately
- Complete profile setup
- Use stable internet connection
