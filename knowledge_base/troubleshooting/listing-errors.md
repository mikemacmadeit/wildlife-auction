# Listing Creation Errors

Common errors when creating listings and how to fix them.

## Common Errors

### Missing Required Fields
- **Error message** - "Required field missing"
- **Solution** - Fill in all required fields
- **Check form** - Review form for empty fields
- **Save draft** - Save and complete later if needed

### Invalid Price
- **Error message** - "Invalid price" or "Price too low"
- **Solution** - Enter valid price (must be greater than 0)
- **Check format** - Use numbers only, no symbols
- **Minimum price** - Some categories have minimums

### Photo Upload Issues
- **Error message** - "Photo upload failed"
- **Solution** - Check file size and format
- **File size** - May be too large (reduce if needed)
- **File format** - Use JPG or PNG format
- **Try again** - Sometimes temporary issue

### Category Mismatch
- **Error message** - "Invalid category" or "Category required"
- **Solution** - Select appropriate category
- **Check selection** - Make sure category is selected
- **Category rules** - Some categories have specific requirements

## Validation Errors

### Compliance Issues
- **Error message** - "Compliance validation failed"
- **Solution** - Review compliance requirements
- **Check attributes** - Ensure all required attributes filled
- **Verify permits** - If required for your category
- **Contact support** - If unsure about requirements

### Location Errors
- **Error message** - "Invalid location" or "Location required"
- **Solution** - Enter valid city, state, ZIP
- **Check format** - Use proper address format
- **Verify location** - Make sure location is correct

## Technical Errors

### Save Failed
- **Error message** - "Failed to save" or "Network error"
- **Solution** - Check internet connection
- **Try again** - May be temporary issue
- **Clear cache** - Browser cache may be issue
- **Contact support** - If persists

### Publish Failed
- **Error message** - "Failed to publish"
- **Solution** - Review all requirements
- **Check status** - Ensure listing is complete
- **Verify account** - Account must be in good standing
- **Try again** - May be temporary issue

## Prevention

### Before Creating
- **Review requirements** - Know what's needed
- **Have photos ready** - Prepare images in advance
- **Complete profile** - Ensure profile is complete
- **Check compliance** - Understand category requirements

### Best Practices
- **Save drafts** - Save work frequently
- **Review before publish** - Check everything before publishing
- **Test photos** - Upload test photo first
- **Read error messages** - They usually explain the issue

## Getting Help

- **Error messages** - Usually explain what's wrong
- **Help articles** - Check relevant help articles
- **Contact support** - If error unclear or persists
- **Provide details** - Include error message when contacting support
