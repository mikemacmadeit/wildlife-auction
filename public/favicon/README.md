# Favicon assets

Add your favicon files here, for example:

- `favicon.ico` (e.g. 32×32 or multi-size)
- `favicon-16x16.png`
- `favicon-32x32.png`
- `apple-touch-icon.png` (180×180)

Then reference them in your app metadata (e.g. `app/layout.tsx` or `app/icon.tsx`) as `/favicon/favicon.ico`, `/favicon/apple-touch-icon.png`, etc.
