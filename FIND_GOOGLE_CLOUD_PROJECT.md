# How to Find Your Google Cloud Project

## Quick Answer

Your Firebase project is: **wildlife-exchange**

This is the same project in both Firebase Console and Google Cloud Console.

---

## Step-by-Step: Find Project in Google Cloud Console

### Option 1: From Firebase Console (Easiest)

1. Go to [Firebase Console](https://console.firebase.google.com/)
2. Select your project: **wildlife-exchange**
3. Click the **gear icon** ⚙️ → **Project settings**
4. Scroll down to **Your project** section
5. You'll see **Project ID**: `wildlife-exchange`
6. Click the **link** next to "Google Cloud Platform" or "Project settings"
   - This will open Google Cloud Console with the correct project selected

### Option 2: Direct to Google Cloud Console

1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Look at the **top bar** - you'll see a project selector dropdown
3. Click the **project dropdown** (shows current project name)
4. If you see **wildlife-exchange**, select it
5. If you don't see it:
   - Click **"Select a project"** or **"New Project"**
   - In the search box, type: `wildlife-exchange`
   - Select it from the list

### Option 3: Check All Projects

1. Go to [Google Cloud Console](https://console.cloud.google.com/)
2. Click the **project selector** at the top
3. Click **"All"** to see all projects
4. Look for: **wildlife-exchange**
5. If you see it, click to select it
6. If you don't see it, the project might be in a different Google account

---

## If Project Doesn't Exist

### Check Firebase Console First

1. Go to [Firebase Console](https://console.firebase.google.com/)
2. Do you see **wildlife-exchange** project?
   - **Yes**: Use Option 1 above to open Google Cloud Console
   - **No**: You might need to create it or you're logged into the wrong account

### Verify You're Using the Correct Google Account

1. Check which Google account you're logged into:
   - Top right corner of Google Cloud Console
   - Top right corner of Firebase Console
2. Make sure it's the **same account** for both
3. If different, sign out and sign in with the correct account
